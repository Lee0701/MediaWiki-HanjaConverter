# MediaWiki-HanjaConverter
[미디어위키][mediawiki]에서 쓸 수 있는 漢字 讀音 變換 擴張機能입니다.

백엔드는 [hanja-reading][hanja-reading] API나 內藏 辭典 中에서 고를 수 있습니다.

[mediawiki]: https://www.mediawiki.org/wiki/MediaWiki
[hanja-reading]: https://github.com/Lee0701/hanja-reading

## 機能
### 漢字 讀音 變換
文書內 모든 漢字語에 `<ruby>` 태그로 讀音을 달아줍니다.

### 使用者 設定
위키에 計定을 만들면 個人別로 讀音을 表示할지 숨길지, 特定 難易度 以上인 글字만 보일지 設定할 수 있습니다.

讀音 位置를 漢字語 側面이나 上端 中에서 定할 수 있습니다.
